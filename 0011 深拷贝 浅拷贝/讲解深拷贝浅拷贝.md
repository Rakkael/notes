1、是什么/区别是？
------
```
浅拷贝：就是拷贝A对象里面的数据，但是不拷贝A对象里面的子对象

深拷贝：会克隆出一个对象，数据相同，但是引用地址不同（就是拷贝A对象里面的数据，而且拷贝它里面的子对象）
```

3、实现方式
------
```
赋值：
var obj2 = obj1{}

浅拷贝：
1、循环赋值
b = {};               
for(let key in a){
    b[key] = a[key];
} 

2、Object.assign()方法
var b = Object.assign({},a) // 注意：当obj1只有一层时，是深拷贝

3、扩展运算符
var b = [...a]

深拷贝：
函数库lodash的_.cloneDeep()方法
var b = _.cloneDeep(a)
```
